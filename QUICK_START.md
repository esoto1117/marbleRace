---

## After Setup - Deploy to Vercel

Once your code is on GitHub:
1. Go to https://vercel.com
2. Click "Add New Project"
3. Import from GitHub
4. Select your `marbleFall` repository
5. Click "Deploy"
6. Your game is live!
